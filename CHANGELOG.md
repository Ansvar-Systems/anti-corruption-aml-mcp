# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-02-28

### Added

- 12 tools covering anti-corruption and anti-money laundering regulatory intelligence
- 256KB database: 8 sources, 123 provisions, 26 FATF country ratings, 10 mutual evaluations, 8 Wolfsberg standards
- FATF 40 Recommendations (2012, updated 2023) with full recommendation text
- UNCAC — United Nations Convention against Corruption (33 articles)
- OECD Anti-Bribery Convention (13 articles)
- EU AML Directives: 4AMLD, 5AMLD, 6AMLD (key provisions)
- EU AML Regulation 2024/1624 (AMLR 2024) key provisions
- GRECO evaluation rounds and recommendations (11 records)
- FATF grey/black list country ratings (26 jurisdictions)
- FATF mutual evaluation results (10 countries)
- Wolfsberg Group standards (8 standards)
- AML risk assessment, obligation mapping, and regime comparison tools
- Dual transport: stdio (npm) + Streamable HTTP (Docker/Vercel)
