# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in the Anti-Corruption & AML MCP, please report it through [GitHub Security Advisories](https://github.com/Ansvar-Systems/anti-corruption-aml-mcp/security/advisories/new).

Do NOT create a public issue for security vulnerabilities.

## Scope

This MCP server provides advisory regulatory intelligence only. It does not:
- Connect to financial institution systems
- Process transaction monitoring data
- Store customer data or suspicious activity reports
- Access sanctions screening APIs
- Handle KYC/CDD documentation

## Supported Versions

| Version | Supported |
|---------|-----------|
| 0.1.x   | Yes       |
